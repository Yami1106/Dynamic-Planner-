## About

A **dynamic motion planning** system for robot navigation in environments with moving obstacles. Implements real-time re-planning strategies to compute collision-free trajectories on the fly.

---

## Features

- Dynamic obstacle detection and avoidance
- Real-time trajectory re-planning
- Multiple planning algorithms (RRT, A*, D* Lite)
- Velocity obstacle and potential field methods
